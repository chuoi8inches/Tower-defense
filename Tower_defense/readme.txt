Projet Tower defense de Nguyen Huu Anh et Cameron Salome - Groupe 3

Les commandes du jeu : 
-Appuyez sur A pour sélectionner tour de flèche
-Appuyez sur B pour sélectionner tour de bombe
-Appuyez sur E pour améliorer une tour existante
-Appuyez sur S pour commencer le jeu //pas complète
-Appuyez sur Q pour quitter le jeu
-Appuyez sur C pour tricher: après, G pour ajouter 100 point d'or, I pour ajouter 100 point de vie, K pour tuer tous les monstre

Nous avons fait en plus: cheatcodes comme indiqué ci-dessus et la génération aléatoire de chemins, 
elle génère avec succès des chemins corrects mais nous ne sommes pas arrivés à faire suivre le chemin par les monstres avec d'autres types de chemin qu'un chemin simple prédéterminé.

La répartition du travail:
-Nguyen Huu Anh: recherche d'images, classes qui gèrent les tours, classes qui gèrent les monstres (notamment monstres volants),classes qui gèrent les projectiles, hitbox et interractions, tests et débuggage, javadoc,concevez le chemin et le backgroud , ...
-Cameron Salome: Gestion des vagues, géneration des chemins aléatoires, classes qui gèrent les monstres (notamment monstres volants), javadoc, ...
